1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego?

    2. - Styl lokalny pozwala nadawać atrybuty wybranym, pojedynczym elementom strony.

  2. Styl lokalny w CSS to sposób nadawania formatowania bezpośrednio w znaczniku HTML danego elementu, za pomocą atrybutu style.

  3. Co może być selektorem?

    4. Selektorem w CSS może być:

    • znacznik
    • klasa
    • identyfikator (id)
  4. Co wipisujemy jako "cechę"? Jakie jest znaczenie średników?

    5. - "Cecha" to inaczej właściwość elementu. Określa co chcemy zmienić w wylądzie.
    - Średnik oddziela kolejne cechy i zamyka jedną deklarację.

  5. Co wipisujemy jako "wartość" (w deklaracji stylu)?

    6. W deklaracji stylu CSS jako "wartość" wpisujemy konkretną informację, jak ma wyglądać dana cecha (właściwość).
