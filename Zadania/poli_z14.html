Temat: Sposoby definiowania (określania) kolorów

Sposoby zapisu kolorów w CSS
Sposób zapisu Przykład Opis
Nazwa koloru red Użycie angielskiej nazwy koloru zdefiniowanej w CSS.
Zapis szesnastkowy (HEX) #FF0000 Kolor określony za pomocą wartości szesnastkowej.
RGB rgb(255, 0, 0) Określenie koloru przez natężenie czerwonego, zielonego i niebieskiego.
RGBA rgba(255, 0, 0, 0.5) RGB z dodatkową wartością przezroczystości (alpha).
HSL hsl(0, 100%, 50%) Kolor określony przez odcień (Hue), nasycenie (Saturation) i jasność (Lightness).
HSLA hsla(0, 100%, 50%, 0.5) HSL z dodatkową wartością przezroczystości (alpha).
