Opis pseudoelementów CSS

1. Pseudoelement :first-line

Pseudoelement :first-line pozwala formatować tylko pierwszy wiersz tekstu znajdującego się w elemencie. Można zmieniać między innymi kolor, rozmiar, krój i styl czcionki.

Składnia:

selektor:first-line
{
    cecha: wartość;
}

Przykład:

p:first-line
{
    color:red;
    font-weight:bold;
}

2. Pseudoelementy :before oraz :after

Pseudoelement :before służy do wstawiania treści przed zawartością elementu.

Pseudoelement :after służy do wstawiania treści po zawartości elementu.

Najczęściej wykorzystuje się właściwość content, określającą tekst lub symbol, który ma zostać dodany.

Składnia:

selektor:before
{
    content:"tekst";
}

selektor:after
{
    content:"tekst";
}

Przykład:

p:before
{
    content:"*** ";
}

p:after
{
    content:" ***";
}

3. Pseudoelement :first-letter

Pseudoelement :first-letter pozwala formatować pierwszą literę tekstu znajdującego się w elemencie.

Jest często wykorzystywany do tworzenia ozdobnych inicjałów na początku akapitów.

Składnia:

selektor:first-letter
{
    cecha: wartość;
}

Przykład:

p:first-letter
{
    color:blue;
    font-size:300%;
    font-weight:bold;
}
