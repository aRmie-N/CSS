Teoria dotycząca zaokrąglania narożników oraz centrowania DIVów

1. Tabela zaokrąglania narożników obszarów bloków np. DIV

Rodzaj zaokrąglenia Składnia CSS
Zaokrąglenie górnego-lewego narożnika selektor { border-top-left-radius: zaokrąglenie; }
Zaokrąglenie górnego-prawego narożnika selektor { border-top-right-radius: zaokrąglenie; }
Zaokrąglenie dolnego-prawego narożnika selektor { border-bottom-right-radius: zaokrąglenie; }
Zaokrąglenie dolnego-lewego narożnika selektor { border-bottom-left-radius: zaokrąglenie; }

2. Zastosowanie oraz uwagi dotyczące zaokrąglania narożników

We wszystkich przypadkach selektorem może być dowolny znacznik, np.:

  • p – akapit
  • h1 – nagłówek
  • div – blok

Parametr "zaokrąglenie" określa promień wyrażony w jednostkach długości lub w procentach.

Możliwe jest podanie:

  • jednej wartości – zaokrąglenie symetryczne,
  • dwóch wartości – pierwsza określa promień poziomy, druga pionowy.

Zaokrąglone narożniki poprawiają estetykę strony internetowej. Element nie musi posiadać obramowania – wystarczy ustalone tło. Należy jednak uważać na zbyt duży promień zaokrąglenia, ponieważ tekst może wychodzić poza obszar elementu.

3. Przykład z instrukcji

border-top-right-radius: 10px;

4. Centrowanie bloków DIV

Aby wyśrodkować blok DIV w poziomie, należy w definicji CSS wpisać: 

margin-left: auto;
margin-right: auto;

Dzięki temu przeglądarka automatycznie wylicza marginesy po lewej i prawej stronie, umieszczając blok na środku strony.
