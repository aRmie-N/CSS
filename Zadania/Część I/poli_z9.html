  2. Gdzie wstawiany jest styl zewnętrzny?

    3. Styl zewnętrzny wstawiany jest w sekcji "head" dokumentu HTML. Odpowiada za to znacznik "link". Może on występować wiele razy w jednym dokumencie.

  3. Najprostsza forma wstawiania stylu zewnętrznego


  4. Ile arkuszy stylów można dołączyć?

    5. Do jednego dokumentu HTML można dołączyć wiele arkuszy stylów. Przy większej liczbie obowiązuje zasada kaskadowości – style wczytane później mogą nadpisywać wcześniejsze.

  5. Co dzieje się w przypadku konfliktów?

    6. Konflikt występuje, gdy te same właściwości są definiowane w różnych arkuszach stylów. Decyduje:

    • kolejność wczytywania plików,
    • specyficzność selektorów,
    • użycie !important.

  6. Jak tworzone są pliki CSS w edytorze tekstowym?

    7. Tworzymy nowy plik tekstowy, zapisujemy go z rozszerzeniem .css, a następnie wpisujemy reguły CSS zgodnie ze składnią.

  7. Jak tworzone są pliki CSS w specjalnym edytorze?

    8. Można używać programów takich jak Visual Studio Code lub Notepad++, które oferują kolorowanie składni, podpowiedzi oraz automatyczne formatowanie.

  8. Jakie rozszerzenie posiada plik zewnętrznego arkusza stylów?

    9. Zewnętrzny arkusz stylów posiada rozszerzenie .css.

  9. Zalety zewnętrznego arkusza stylów:
    • Oddzielenie treści od wyglądu strony.
    • Możliwość użycia jednego pliku CSS na wielu stronach.

  10. Uwagi na temat pisania w CSS:
    • Poprawna składnia (średniki i nawiasy klamrowe).
    • Stosowanie wcięć dla czytelności.
    • Unikanie powielania kodu.
    • Komentowanie ważnych fragmentów.
